# ‚úÖ **1. The Universal Home Textile Cost Formula (Same for All Scales)**

Every home textile product‚Äîbedsheets, curtains, towels, cushions‚Äîuses **the same cost structure**, no matter the company size.

### **Master Formula**

    Total Cost = 
    (Material Cost)
    + (Processing Cost)
    + (Cutting & Sewing Cost)
    + (Finishing & Packing Cost)
    + (Overheads)
    + (Wastage Allowance)

If a department doesn‚Äôt exist in a small company, that section is simply **zero** or **merged**.

***

# ‚úÖ **2. How Each Department Feeds Into Cost**

These inputs go into the formula:

### **Raw Material / Purchase / Stores**

*   Fabric/Yarn price
*   Trims (zips, labels, thread)
*   Packaging (polybag, carton)

### **Weaving / Knitting**

*   Loom time
*   Efficiency
*   Picks per inch / GSM impact

### **Dyeing / Printing**

*   Dye chemical usage
*   Print coverage
*   Batch weight

### **Cutting**

*   Marker efficiency
*   Fabric utilization
*   Wastage

### **Stitching / Tailoring**

*   SMV (time)
*   Labor rate
*   Line efficiency

### **Quality**

*   Reject %
*   Rework time

### **Finishing / Packing**

*   Ironing / folding time
*   Packing material usage

### **Overheads (General Factory + Admin)**

*   Electricity
*   Rent
*   Supervisor salaries
*   Machine depreciation

Some companies track these in detail; some just use a %.

# üü¶ **1) Large-Scale Home Textile Industry (Export Factories)**

**Highly detailed costing**  
**Each department feeds separate cost data**

### **How it works**

*   Every activity has **standard time**, **machine rate**, **labor rate**, **overhead pool**.
*   Uses **BOM + Routing + Overhead absorption**.
*   Data comes from:  
    Weaving, Dyeing, Printing, Cutting, Stitching, QC, Packing, HR, IE, Accounts, Stores, Transport, Documentation.

### **Large Scale Formula**

    Material = Œ£(BOM √ó Price √ó Wastage%)
    Process = Œ£(Operation time √ó Rate/min √ó Efficiency)
    Overhead = Œ£(Department pool √∑ Driver units)
    Wastage = Total √ó Wastage%

### **Characteristics**

*   Most accurate
*   ERP‚Äëintegrated
*   Real usage vs. standard usage tracking

***

# üü© **2) Medium-Scale Home Textile Industry**

**Partially detailed costing**  
Departments may be merged (ex: Cutting+Stitching, QA+Packing)

### **How it works**

*   Uses **BOM** but routing is simpler.
*   Uses **SMV √ó labor rate** for stitching.
*   Overheads often applied as a simple percentage (10‚Äì18%).

### **Medium Scale Formula**

    Material = BOM √ó Price √ó 1.X wastage factor
    Process = (Cutting SMV + Stitching SMV + Finishing SMV) √ó Labor rate
    Overhead = % of manufacturing cost

### **Characteristics**

*   Balanced accuracy
*   Easier to maintain
*   Useful for domestic manufacturers

***

# üüß **3) Small-Scale Textile Unit (Local factories / job-work units)**

**Simplified costing**  
Most work done by experience, not detailed measurement.

### **How it works**

*   No SMV study; time estimated by supervisor.
*   Overhead added as flat amount per piece.

### **Small Scale Formula**

    Material = Fabric Cost/meter √ó consumption + trims
    Labor = Per-piece stitching rate + cutting rate
    Dye/Print = Per-meter fixed job-work rate
    Overhead = Fixed amount/piece (e.g., ‚Çπ3‚Äì‚Çπ10)

### **Characteristics**

*   Quick calculation
*   Based on ‚Äúmarket rate‚Äù
*   Good for job-work costing

***

# üü® **4) Extra Small / Individual (Tailors / Small Home Units)**

**Very basic costing**

### **How it works**

*   Material bought from market
*   Stitching charged per piece
*   No department cost splitting

### **Extra Small Formula**

    Material Cost = Fabric (meters √ó price/meter)
    Labor Cost = Stitching charge per piece
    Packing Cost = Simple polybag
    Profit Margin = Add 15‚Äì25%

### **Characteristics**

*   Minimum data
*   No overhead calculation
*   Ideal for Etsy/small shop/home business
